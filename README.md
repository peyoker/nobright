# brightnesstoogle
windows application to change monitor brightness with one programable key



-compiled in vscode with C# dev kit extension.

-.net 9 sdk is needed

-add nuget source to dontnet (dotnet nuget add source "https://api.nuget.org/v3/index.json" -n "nuget.org")


to execute need .net 9 runtime desktop https://dotnet.microsoft.com/es-es/download/dotnet/9.0
